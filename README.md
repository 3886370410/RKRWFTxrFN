# 项目简介

本项目致力于通过基于深度学习方法的高光谱与多光谱图像融合技术，提供一种比较算法研究平台。系统集成了7种不同的融合方法，并配备了相应的代码以及3个预处理的实际数据集，助力科研人员对融合算法进行有效对比与分析。技术特点包括多样化的融合技术实现以及便捷的数据集接入，为图像融合领域的研究与实践提供支持。


## 核心功能
- 提供7种基于深度学习的高光谱与多光谱图像融合算法实现
- 集成完整的代码库，支持算法对比分析
- 包含3个预处理好的实际数据集，可直接用于算法测试
- 构建算法研究平台，便于科研人员进行融合方法对比

## 适用场景
- 高光谱与多光谱图像融合算法研究
- 不同融合方法性能比较分析
- 遥感图像处理技术研究
- 图像融合领域学术实验与验证

## 技术特点
- 基于深度学习的多种融合技术实现
- 提供标准化的数据集接入接口
- 支持算法间公平对比的实验环境
- 完整的代码实现确保研究可复现性
